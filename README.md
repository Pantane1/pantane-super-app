# Pantane Super App

**Tagline:** One App. Every Hustle. African Power.

## Description
Pantane is an all-in-one African business app designed for Kenyan and African entrepreneurs. It combines payments, marketplace, business management, and learning tools into a unified platform. Empower users to earn, learn, trade, and grow in one digital home.

## Core Sections
- **PantaneBiz**: Manage sales, expenses, and profits.
- **PantaneBiz Pro**: Add staff, track analytics, and generate invoices.
- **PantaneBiz Market**: Buy and sell products, connect with verified sellers.
- **PantaneBiz Pay**: Send money, receive payments, and pay bills via M-Pesa or wallet.
- **PantaneBiz Learn**: Access business courses, mentorship, and certificates.
